Задание №4 — класс «Множество» (Set)
Необходимо реализовать обобщённый класс «Множество», который хранит уникальные элементы и предоставляет методы для работы с ними.

Методы множества:
Add(T item) - Добавить элемент во множество (если элемент существует, ничего не происходит) \
Remove(T item) - Удалить элемент из множества (если элемента нет, ничего не происходит) \
Clear() - Очистка множества (удаление всех элементов) \
GetElements() - Получить массив всех элементов множества \
Contains(T item) - Проверить наличие элемента во множестве (возвращает true/false) 

Язык программирования - C# \
Платформа - .NET 10.0 \
UI фреймворк - Avalonia UI \
MVVM Toolkit - CommunityToolkit.Mvvm
